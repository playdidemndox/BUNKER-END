# Бункер: Поколение Альфа — Mini App

## Описание
Это веб-версия настольной игры "Бункер: Поколение Альфа" для Telegram Mini App. 
Игра поддерживает до 10 игроков, раздаёт случайные карточки персонажей, ведёт раунды, открытие карт и голосование.

---

## Структура проекта
bunker-miniapp/
├─ server.js # Сервер на Node.js
├─ package.json # Зависимости и команды запуска
├─ README.md # Инструкция
└─ public/
├─ index.html # Мини-приложение
├─ style.css # Стили для карточек и интерфейса
├─ script.js # Логика игры и взаимодействие с сервером
└─ cards.json # Все 321 карта персонажей и особых условий
